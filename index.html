How to View Your Cricket Statistics

Since this is a static page hosted on GitHub Pages, it cannot execute Python scripts directly. To view your cricket statistics, please follow these steps:

1. Local Setup

  • Clone the repository:
    git clone https://github.com/Avinash9271/ipl-dv.git
    cd ipl-dv
  • Install required dependencies:
    pip install -r requirements.txt

2. Run the Script

  • Option 1 - Run with API key as argument:
    python cricket_scorer.py YOUR_API_KEY
  • Option 2 - Set environment variable and run:
    export CRICAPI_KEY=YOUR_API_KEY
    python cricket_scorer.py

3. View Results

The script will:

  • Fetch live match data from CricAPI
  • Process ball-by-ball statistics
  • Calculate scores using the following system:
    • Base score: 4 points
    • Batting: 1 point per run, bonus for 50s and 100s
    • Bowling: 20 points for bowled/LBW/caught & bowled, 16 for other wickets
    • Fielding: 8 points each for catches and run outs
    • 1.3x multiplier for highest scorer in each match
  • Display team owner and player statistics in a formatted table
